FACTS Benefits of GrowDirector
  • Easy to use

    Lights, but not only, we can control any electrical equipment you might have. Its wireless and modular system can be used to connect up to 64,000 sensors and devices.

  • GrowDirector PRO can do it ALL

    It's great for hydroponics or whatever growing media you want to use. All kinds of sensors: temperature, humidity, CO2, VPD, EC, pH, DO, ORP, PAR, and many more.

  • Open API

    Compatible with other greenhouse software and can share data between them.

  • Exclusive patented innovation

    A machine learning system that can prevent growers’ mistakes and predict the environment.

  • Access from anywhere
